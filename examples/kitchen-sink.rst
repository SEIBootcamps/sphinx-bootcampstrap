Kitchen Sink

This extension is meant to be used alongside a theme.

---------------

-------------

#. Item 3

a. Item 1

c. Item 3

Comparing Things
================

You can use ``.. compare::`` to display containers side-by-side.

.. compare::

   .. container::

      Here's a paragraph.

   .. code-block:: python

      # Here's some code
      def say_hello():
          print("Hello, world!")

It supports an arbitrary number of containers.

.. compare::

   .. container::

      Column 1

   .. container::

      Column 2

   .. container::

      Column 3

   .. container::

      Column 4
